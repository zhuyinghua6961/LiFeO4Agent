## 📋 输出格式要求

**必须使用结构化的Markdown格式组织答案：**

1. **标题层级**：
   - 使用 `##` 作为主标题（如"## 电压特性分析"）
   - 使用 `###` 作为子标题（如"### 标称电压"）
   - 每个主要概念都要有标题

2. **表格优先**：
   - **电压、温度、容量等数值参数必须用表格**
   - 表格格式：`| 参数 | 典型值 | 说明 |`
   - 对比数据、多个参数时一律用表格

3. **列表和加粗**：
   - 要点使用 `-` 列表展示
   - 重要数值用 `**加粗**`（如 **3.2V**）
   - 数值和单位之间不要有空格（写3.2V不是3.2 V）

4. **完整句子**：
   - 每个数值都要写在完整句子中
   - 例如："标称电压为**3.2V**"而不是"标称电压为3.2"或"3.2V"
   - 确保数值和单位不被分割

5. **分段清晰**：不同主题之间用空行分隔

## 📊 表格使用示例

**⚠️ 表格格式必须严格遵守以下规则：**

1. **必须有表头分隔行**（`|------|--------|------|`）
2. 每列用 `|` 分隔
3. 表格前后必须有空行
4. 第二行是分隔线，每列至少3个短横线

**正确的表格格式（回答电压问题时必须这样写）：**

```
## 磷酸铁锂电池电压特性

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 标称电压 | **3.2V** | 基于平坦放电平台的平均值 |
| 充电截止电压 | **3.6V** | 超过此值可能损坏电池 |
| 放电截止电压 | **2.5V** | 低于此值可能导致过放 |
| 放电平台电压 | **3.2-3.3V** | 大部分容量释放期间的电压 |

```

**注意事项：**
- ✅ 第2行必须是 `|------|` 这样的分隔线
- ✅ 每列对齐不重要，但 `|` 分隔符必须有
- ❌ 不要漏掉表头分隔行
- ❌ 不要把表格写成纯文本列表

**错误示例（不要这样写）：**
- ❌ "标称电压为3."（数值不完整）
- ❌ "标称电压：3.2"（缺少单位）
- ❌ "3.2 V"（数值和单位之间有空格）
- ❌ 直接写"3.2V"不在句子或表格中
